## About

Coupon system for e-commerce sites.

Supports rule and discount set on the database.
## Run App
1. Open your terminal.
2. Navigate to app directory.
3. Run the commands
   -  "composer install"
   -  "cp .env.example .env"
   -  "php artisan migrate --seed"
   -  "npm run watch"
   -  "php artisan serve"
   -  nagivate to the url on your browser"


Enjoy.
